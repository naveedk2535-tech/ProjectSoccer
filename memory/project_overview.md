Building a soccer prediction engine with a modern dark-themed dashboard (Bloomberg Terminal meets FotMob aesthetic).

**What it does:**
- Predicts match outcomes using Poisson, Dixon-Coles, Elo, XGBoost models + sentiment
- Compares model predictions against bookmaker odds to find value bets
- Displays everything in an interactive dashboard — no CSV exports
- Covers Premier League first, then expanding to 7 leagues total

**Why:** User wants a personal tool to identify value bets by combining statistical models with market odds, similar to how a stock trading dashboard works.

**How to apply:** Every feature should serve the dashboard. The end product is a visual, interactive tool — not scripts or exports.

**Tech:** Flask + SQLite + Tailwind/Alpine.js + Chart.js, deployed on PythonAnywhere (/home/zziai38)
