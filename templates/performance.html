Model Performance -- Help

What is this page?

This is proof the model works. We ran it on historical matches where we already know the results, and measured how well it predicted them.

Accuracy

What % of matches did each model predict correctly. Random guessing would be ~33% (three outcomes: H/D/A).

Brier Score

A more sophisticated accuracy measure. Lower = better. 0 = perfect predictions. 0.25 = random guessing.

ROI

If you had bet every value bet the model flagged, what would your return be?
